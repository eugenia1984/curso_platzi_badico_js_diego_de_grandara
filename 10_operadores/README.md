| Simbolo	| Descripcion |

## Operadores de incremento y decremento

Para incrementar el valor de una variable en 1:

```JavaScript
let edad = 10;
edad++;
console.log(edad);   // 11
```
Si quiero incrementar el valor de la variable en 2 o en más: 

```JavaScript
edad += 2;
console.log(edad); // 13
```

Para decrementar utilizo: **--** ó **-=**
